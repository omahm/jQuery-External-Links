ExternalLinks

This plugin appends an image link to the end of any external links in your content and gives you complete control over it's look and behaviour.

The link target, image, image dimensions, class and title can be changed by passing in your own settings, see the options and examples sections for details.

Usage

  1. Add a script reference to both jQuery and externalLinks javascript files.
  2. Add an external script or inline script calling jQuery, passing in your selector and then calling externalLinks().
                                                                  
<html>                                                                  
 <head>                                                                  
 <script type="text/javascript" src="jquery.js"></script>          
 <script type="text/javascript" src="jquery.externallinks.js">                                         
 <script>
    $('body').externalLinks();                                     
 </script>                                                               
 </head> 
 <body>
   -- your content --  
 </body>
</html>

Options

By default the plugin will append a 10px X 10px Data URI image link with a class name of 'extlink' and a title of 'Open this link in a new window/tab' to any external link and the clicked link opens in a new tab or window depending on the browser preferences.
Property Details Default
target Target window for the link.
e.g. _self, _blank, _top etc.		 _blank
image The image to display, can be either a data URI or path to an image file. data uri (IE8+)
imageWidth Width in pixels of the link image.
Whole numbers only		 10
imageHeight Height in pixels of the link image.
Whole numbers only		 10
className Name to add to the class attribute (leave blank for no class attribute). extlink
title Tooltip text to display on mouse over. Open this link in a new window/tab

Examples

Defaults settings

$(document).ready(function() {
    $('body').externalLinks();
});

Open external links in this window, use a relative image file of size 8px X 8px. Use a class name of 'external-link' and don't set a title attribute.

$(document).ready(function() {
    $('body').externalLinks({
        target      : '_self',
    	image       : '/images/extlink.png',
    	imageWidth  : '8',
    	imageHeight : '8',  
    	className   : 'external-link',
    	title       : ''	
    });
});
